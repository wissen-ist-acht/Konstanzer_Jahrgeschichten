About the Project

A digital representation of manuscripts containing the Konstanzer Jahrgeschichten

This page offers an overview of the manuscripts that contain the so-called Konstanzer Jahrgeschichten with corresponding transcriptions. The transcriptions are diplomatic and are encoded in XML-TEI. The transcriptions are licensed under CC BY 4.0.

The data can be found on this repository.

It is still a work in progress, new transcriptions will be uploaded regularly.

Features and technical information

  • TEI encoding: All data is encoded following TEI P5 guidelines for scholarly digital editions
  • CETEIcean integration: Native display of TEI XML in the browser
  • TEI P5: Text Encoding Initiative guidelines for scholarly digital editions
  • CETEIcean: JavaScript library for displaying TEI XML in web browsers
  • Bootstrap 5: CSS framework for responsive design
